# Lifehackchecker
The purpose of the app is to help users to be more informed  and aware of misleading information. 

 

The layout of the app is simple and clear to the users .  

The heading is large and easy to read . 

 

The dark background with a purple button creates visibility .  

 

The user chooses whether the statement displayed  is true or false ,then the app will display whether the user chose the correct answer   

 

Once the user has completed all the questions ,the app will display their total score .
